What is HTML Boilerplate

Boilerplate is HTML's template, that can be reused multiple times. It usually starts with document type: <!DOCTYPE html>. Doctypes tell browser which version html the code is written in.

Tag head contains useful information about the page. It helps search engines to locate the web page or it can contain additional metadata via meta tag.
For Example:

  • <meta charset="UTF-8">
  • <meta name="description" content="THis is notes for html.">

And then comes body tag. Body tags contain every content of the webpage that's being rendered right in front of you.

How to structure HTML text

Indents are required!

Any tag's child must start from new indet!
And sibling tags go in same indent!

New tag <paragraph>.
This tag is used for paragraphs and texts.

also you can stylize text with these tags:
<strong> and <b> for bolding the text inside the open and close tags.
<em> and <i> for emphasizing and italicizing the text

Lists in HTML

There are 2 types of lists:

<ul> for unordered list and <ul> for ordered list

Inside lists you need another tag for each item in the list : <li>
You can nest lists inside lists.

Image in HTML

To render image you need this tag:
<img src="source of the image file" alt="explanation of image when image is not available">

Links and Anchor Tags

To create a hyperlink text or file you need an anchor tag:
<a href="link or file"> text or input for link</a>

Tables in HTML

To create a table use:
<table> -Table
 <tr> -Table row
  <td> -Table data/cell
 </tr> -Table row end
</table> -Table end
