CSS - Cascading Style Sheets

CSS is a tool for styling html.
There are 3 kinds of css code:

Inline
Internal
External



Inline CSS code

Inline css codes are written inside some tag.

For Example: < body style="background-color:blue;" >

Colors inside css can be: name of the color, #Hexadecimal code, RGB code, etc


Internal CSS code

For internal code you need another tag:
<%style%>
selector{property:value;}
h2{color: red;}
<%/style%>


External CSS code

For external code you need another file: Name.css

To link your new css file to you html file you need <%link rel="stylesheet" href="location"%> In your head tag.

Id Vs Classes

You can aim for specific tag in your css code using either id or classes

<% img class="name" %> and in CSS code: .name{}

<% p id="name" %> and in CSS code: #name{}

Tags can have multiple classes but only one id. And one particular id can only be used once in one page.


FavIcons and DIVS in HTML

Favicons are tiny icon for your web page in the browser tab

Divs are containers for some tags and manipulate them using one class or id

The Box Model

Every elements are just boxes in browser

There is a hierarchy for this:

1.Margin
2.Border
3.Padding
4.Tag itself


Display propery

There are 4 display properties:

1.Block property - this property makes elements to block every other objects in it's line. It means these element are alone in one line.
2.Inline property - this property does not block other elements they stay in one line. But you can't change width or height of an element
3.Inline-block - this property has best of both previous two properties. They can stay in one line and still can change widht and height.
4.None - this makes your element disappear.
